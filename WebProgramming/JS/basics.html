Variables in JS

3 ways to store the data in JS:

  1. var keyword:Old school of creating variables.
  2. let keyword: New way of declaring variables and said to be powerfull
  3. const keyword: For storing data which does not change
Js does not have any data type in it. Internally it differenciates the types of data as number, string, boolean, object, undefined.
