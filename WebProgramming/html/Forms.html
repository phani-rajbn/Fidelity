Forms are pages that contain input tags to take inputs from the user for the Web application. It is typical to a form that U see in a day to day life. The inputs will provide the scope for the user to type things, select from a list, select an option or a checkbox, select a range, date and many other things that a user wishes to send information to the application. All user inputs are expected to be put under the form tag.
